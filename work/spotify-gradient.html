Spotify Gradient

Spotify Gradient is a web app that experiments with the Spotify API to sort playlists by album cover. This program sorts a user's playlists based on the dominant colors of their albums' artwork. Spotify Gradient is live and can be visited at


https://spotify-gradient.herokuapp.com/


As I started creating this program, I set out to play with the Spotify API and build something creatively stimulating. Knowing there were already a slew of Spotify attachment apps, my goal was to make something I had never seen before. Spotify Gradient is designed to emphasize the visual elements of a predominantly auditory art. I built an AngularJS app that, for any given playlist, extracts the most salient color from each album's artwork and orders the tracks by their color, using RGB & HSV values, to create a gradient within the playlist.

Process


Spotify Gradient uses AngularJS for back-end work and communication with the Spotify API; I considered building this program with both Vue.js and React before deciding on AngularJS. In order for my program to work, I needed to access Spotify authentication and playlist & user data. Using the available documentation and sample code, I developed the basic functionality for Spotify Gradient.

To sort the track covers by color, I used the existing ColorThief library to extract the dominant color from each track cover image; this yielded RGB values for each color, which I converted to HSV values.


HSV values take up a 3-dimensional space. Playlists can only accommodate one dimension, so covers that are darker or lighter than a certain saturation/value threshold are sorted separately.

In between these thresholds, the covers are sorted by hue and then by value. Outside of these thresholds (predominantly black & white images), the covers are sorted by value and then by hue.

The layout for this program is based on the original Spotify aesthetics and uses similar colors, fonts, and structure. I focused on making the interface clean, intuitive, and simple, so as to highlight the sorting functionality that the program provides.


With the CSS formatting and the AngularJS back-end work complete, I used Heroku to deploy my program to the web. Heroku was a good fit for this project because it offered a simple deployment environment; the point of Spotify Gradient was to experiment with the Spotify API, so there was no need for a complex deployment platform.
