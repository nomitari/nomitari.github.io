Spotify Gradient

Spotify Gradient is a web app that experiments with the Spotify API to sort playlists by album cover. This program sorts a user's playlists based on the dominant colors of their albums' artwork. Spotify Gradient is live and can be visited at


https://spotify-gradient.herokuapp.com/


As I started creating this program, I set out to play with the Spotify API and build something creatively stimulating. Knowing there were already a slew of Spotify attachment apps, my goal was to make something I had never seen before. Spotify Gradient is designed to emphasize the visual elements of a predominantly auditory art. I built an AngularJS app that, for any given playlist, extracts the most salient color from each album's artwork and orders the tracks by their color, using RGB & HSV values, to create a gradient within the playlist.
