Dance Visualizer

This dance visualizer program allows the user to create and view a dance sequence as an ASCII animation. Developed as an exploration of ASCII art, this project aims to interactively intersect dance and programming. The user of this program can make selections from a library of dance moves and genres, optionally defining time spans for each move they select. The final sequence is then played out on the screen in ASCII characters.
